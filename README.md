# 📊 Retail Sales Analysis Dashboard

## Overview
This project is an interactive Power BI dashboard developed using the Sample Superstore dataset to analyze retail sales performance and generate business insights.

## Technologies Used
- Power BI
- DAX
- Microsoft Excel

## Dashboard Features
- Sales Analysis
- Profit Analysis
- Regional Performance
- Customer Segment Analysis
- Product Category Analysis
- KPI Cards
- Interactive Slicers and Filters

## Key Insights
- Analyzed sales and profit across different regions.
- Compared customer segments and product categories.
- Built interactive visualizations to support data-driven decision-making.
